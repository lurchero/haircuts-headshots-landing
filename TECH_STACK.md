# Tech Stack

## Framework
- **Next.js 14** (App Router)
- **React 18**
- **TypeScript** (strict mode)

## Styling
- **Tailwind CSS 3.4+**
- **tailwind-merge** — for conditional class merging
- **clsx** — for conditional classnames

## Animations
- **Framer Motion** — for scroll animations and micro-interactions

## Icons
- **Lucide React** — consistent, minimal icon set

## Fonts
- **Inter** (Google Fonts) — primary font
- **Font optimization** via `next/font`

## Development
- **ESLint** — code linting
- **Prettier** — code formatting
- **Node.js 18+**

## Deployment Target
- **Vercel** (recommended) or any Node.js hosting

## Package Manager
- **npm** or **pnpm**

---

## Commands

```bash
# Create project
npx create-next-app@latest landing-page --typescript --tailwind --app --eslint

# Install dependencies
npm install framer-motion lucide-react clsx tailwind-merge

# Development
npm run dev

# Build
npm run build
```
